06_01 Dropdown Demo

Wrap the <button> and the <ul> with a container <div class="dropdown">

the <button> needs a data-toggle="dropdown" attribute while the <ul> needs a class="dropdown-menu"

The down arrow was created by using span class="caret" inside the <button>

Example:

  • Common
  • Cat
  • Dog
  • Fish
  • Birds
  • Exotic
  • Amphibian
  • Reptile
  • Other

To add the header inside a dropdown list, simply give the li class="dropdown-header"

To add a divider, simply give the li class="divider" and add attribute role="seperator"
